📌 วิธีตั้งค่า Google Sheets API:
1. เปิด Google Cloud Console
2. เปิดใช้งาน "Google Sheets API"
3. สร้าง API Key ใน Credentials
4. ตั้งค่า Spreadsheet เป็น "Anyone with the link can view" หรือใช้ Service Account
5. วาง API Key และ Spreadsheet ID ด้านบน แล้วกดบันทึก

⚙️ ตั้งค่าหน่วยงานและปีงบประมาณ
📌 วิธี Deploy Apps Script (ทำครั้งเดียวต่อสำนักงาน):
1. เปิด Google Sheet → Extensions → Apps Script
2. วางโค้ดจาก Code.gs
3. Deploy → New deployment → Web app
4. Execute as: Me  |  Who has access: Anyone
5. Copy URL → วางใน "Apps Script Web App URL" ด้านบน → บันทึก
